[[member-api]]
== Member API

=== 모듈 책임

Member 모듈은 사용자 계정, 자체 로그인, 소셜 로그인, 토큰, 학교 선택, 학교 이메일 인증 흐름을 담당합니다.

* 회원가입과 프로필 관리
* 자체 로그인과 소셜 로그인
* Access Token / Refresh Token 관리
* 학교 목록 조회
* 학교 이메일 인증 요청과 확인
* 사용자 신뢰 점수 조회와 갱신

=== API 목록

[cols="1,2,3",options="header"]
|===
|기능 |메서드 / 경로 |설명

|학교 목록 조회
|`GET /api/v1/schools`
|활성 학교 목록을 조회합니다. `keyword`로 학교명을 검색할 수 있습니다.

|회원가입
|`POST /api/v1/members/signup`
|신규 사용자를 생성하고 학교 인증 대기 상태를 만듭니다.

|로그인
|`POST /api/v1/auth/login`
|이메일과 비밀번호로 로그인하고 토큰을 발급합니다.

|토큰 재발급
|`POST /api/v1/auth/token/refresh`
|Refresh Token으로 Access Token을 재발급합니다.

|내 프로필 조회
|`GET /api/v1/members/me`
|현재 로그인한 사용자의 프로필을 조회합니다.

|내 프로필 수정
|`PATCH /api/v1/members/me`
|현재 로그인한 사용자의 프로필 정보를 수정합니다.

|학교 이메일 인증 요청
|`POST /api/v1/school-verifications/request`
|학교 이메일로 인증 코드를 발송합니다.

|학교 이메일 인증 확인
|`POST /api/v1/school-verifications/confirm`
|인증 코드를 검증하고 학교 인증 상태를 저장합니다.

|신뢰 점수 조회
|`GET /api/v1/members/{memberId}/trust-score`
|사용자의 신뢰 점수를 조회합니다.
|===

=== Signup APIs

=== Auth APIs
