[[market-api]]
== Market API

=== 모듈 책임

Market 모듈은 캠퍼스 중고거래 판매글과 상품 탐색 흐름을 담당합니다.

* 판매글 등록, 조회, 수정, 삭제
* 상품 정보와 카테고리 관리
* 판매글 이미지 업로드
* 관심/찜 등록과 해제

=== API 예정 목록

[cols="1,2,3",options="header"]
|===
|기능 |예상 메서드/경로 |설명

|판매글 등록
|`POST /api/market/posts`
|판매글과 상품 정보를 등록합니다.

|판매글 목록 조회
|`GET /api/market/posts`
|검색, 카테고리, 정렬 조건으로 판매글 목록을 조회합니다.

|판매글 상세 조회
|`GET /api/market/posts/{postId}`
|판매글 상세 정보와 판매자 요약 정보를 조회합니다.

|판매글 수정
|`PATCH /api/market/posts/{postId}`
|판매글 내용을 수정합니다.

|판매글 삭제
|`DELETE /api/market/posts/{postId}`
|판매글을 삭제합니다.

|이미지 업로드
|`POST /api/market/images`
|판매글에 사용할 이미지를 업로드합니다.

|카테고리 조회
|`GET /api/market/categories`
|판매글 카테고리 목록을 조회합니다.

|관심/찜 등록
|`POST /api/market/posts/{postId}/likes`
|판매글을 관심 목록에 추가합니다.

|관심/찜 해제
|`DELETE /api/market/posts/{postId}/likes`
|판매글을 관심 목록에서 제거합니다.
|===
