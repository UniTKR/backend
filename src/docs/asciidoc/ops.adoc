[[ops-api]]
== Ops API

=== 모듈 책임

Ops 모듈은 운영, 신고, 관리자 처리 흐름을 담당합니다.

* 신고 생성과 조회
* 신고 처리
* 관리자 사용자 조회
* 사용자 제재와 제재 해제

=== API 예정 목록

[cols="1,2,3",options="header"]
|===
|기능 |예상 메서드/경로 |설명

|신고 생성
|`POST /api/reports`
|사용자, 판매글, 채팅 등 신고 대상을 접수합니다.

|신고 목록 조회
|`GET /api/admin/reports`
|관리자가 신고 목록을 조회합니다.

|신고 상세 조회
|`GET /api/admin/reports/{reportId}`
|관리자가 신고 상세 내용을 조회합니다.

|신고 처리
|`POST /api/admin/reports/{reportId}/resolve`
|관리자가 신고 처리 결과를 기록합니다.

|관리자 사용자 조회
|`GET /api/admin/members`
|관리자가 사용자 목록을 조회합니다.

|사용자 제재
|`POST /api/admin/members/{memberId}/sanctions`
|사용자에게 제재를 적용합니다.

|사용자 제재 해제
|`DELETE /api/admin/members/{memberId}/sanctions/{sanctionId}`
|사용자 제재를 해제합니다.
|===
